# Paradise Nursery Shopping Application

Paradise Nursery is a React and Redux shopping cart application for an online plant store. The application allows users to browse houseplants by category, view product names, images, and prices, add plants to a cart, adjust quantities, remove items, and view dynamic cart totals.

## Project Features

- Landing page with the Paradise Nursery company name and a Get Started button
- About page describing the company
- Product listing page with houseplants grouped into categories
- Add to Cart buttons for each plant
- Dynamic cart icon count in the navigation bar
- Shopping cart page with item quantity controls, delete buttons, total quantity, and total cost
- Checkout button showing a Coming Soon message
- Continue Shopping button linking back to the product listing page

## Technologies Used

- React
- Redux Toolkit
- React Redux
- Vite
- CSS
